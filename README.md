# Блогикум

Проект «Блогикум» - это многофункциональная блог-платформа, разработанная на Django, которая предоставляет пользователям возможность публиковать посты, комментировать записи и взаимодействовать с контентом.

## Функции

### Управление публикациями
- **Создание публикаций**
- **Редактирование публикаций**
- **Удаление публикаций**
- **Отложенные публикации**
- **Категории и местоположения**

### Система пользователей
- **Регистрация и аутентификация**
- **Профили пользователей**
- **Редактирование профиля**
- **Управление доступом**

### Комментирование
- **Добавление комментариев**
- **Редактирование комментариев**
- **Удаление комментариев**

### Навигация и организация контента
- **Пагинация**
- **Категории**

## Требования
- Python 3.13
- Django 3.2+
- Зависимости, указанные в requirements.txt

## Запуск
- `python -m venv venv`
- `venv\Scripts\activate`
- `pip install -r requirements.txt`
- `python manage.py migrate`
- `python manage.py createsuperuser`
- `python manage.py runserver`
